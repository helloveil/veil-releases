# Veil — Privacy-First Meeting Recorder

Download the latest version of Veil for macOS.

## Download

**[Download Veil for macOS](https://github.com/KarmaFounder/veil-releases/releases/latest)**

Requires macOS 14.0+

## Installation

1. Download the DMG
2. Drag Veil to Applications
3. Right-click Veil.app > Open (first launch only)

## Auto-updates

Veil checks for updates automatically. You'll be notified when new versions are available.
